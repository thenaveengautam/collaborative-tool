## ğŸ“ project summary

A modern web application for team project management with Kanban-style task boards, real-time collaboration, and intuitive user interface.
Features

User Authentication - Secure login and registration system
Project Management - Create and manage multiple projects with team members
Kanban Board - Drag-and-drop task management with 4 columns (To Do, In Progress, Review, Done)
Task Management - Create, edit, assign, and prioritize tasks
Real-time Notifications - Stay updated with project activities
Comments System - Collaborate with team members on tasks
Responsive Design - Works on desktop, tablet, and mobile devices
